### 指标计算
- 计算出各个省的成交量总额（结果保存到MySQL中）             
- 计算每个省城市成交量的top3（结果保存到MySQL中）
- 计算每个商品分类的成交总额，并按照从高到低排序（结果保存到MySQL中）
- 构建每一个用户的用户画像，就是根据用户购买的具体商品，给用户打上一个标签，为将来的商品推荐系统作数据支撑
